Teleprompter oder Karaoke Animation

Hier ist ein einfacher Karaoke-Effekt mit etwas CSS und JavaScript.

The only way to do great work is to love what you do. — Steve Jobs

Tipp: Drücken Sie Shift + Cmd + P oder Tap auf einem Touchscreen um die Animation erneut abzuspielen.

Der Code für den Karaoke-Effekt ist unten aufgeführt. Sie können ihn kopieren und in Ihre eigenen Projekte einfügen.

CSS

:root {
    --highlight-color: #1068af; /* or your preferred color */
    --default-color: inherit;
    --faded-opacity: 1;
    --highlight-opacity: 1;
}

.readme span {
    opacity: var(--faded-opacity);
    color: var(--default-color);
    animation-name: highlight;
    animation-iteration-count: 1;
    animation-fill-mode: forwards;
    transition: text-shadow 0.3s;
}

.readme span.emphasized {
    font-weight: bold;
}

@keyframes highlight {
    0% {
        opacity: var(--highlight-opacity);
        color: var(--highlight-color);
        text-shadow: 0 0 0px var(--highlight-color), 0 0 0px #fff;
    }
    30% {
        opacity: var(--highlight-opacity);
        color: var(--highlight-color);
        text-shadow: 0 0 2px var(--highlight-color), 0 0 10px #fff;
    }
    80% {
        opacity: var(--highlight-opacity);
        color: var(--highlight-color);
        text-shadow: 0 0 4px var(--highlight-color), 0 0 18px #fff;
    }
    100% {
        opacity: var(--faded-opacity);
        color: var(--default-color);
        text-shadow: none;
    }
}
